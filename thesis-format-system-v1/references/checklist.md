# Final checklist

Before completion, confirm all of these:

- the requirements file was fully reviewed
- the working thesis is `docx`
- a backup exists
- margins and page setup match the rules
- headers and footers are correct
- page-number logic is correct across front matter and body
- heading hierarchy is correct
- navigation works
- the TOC is automatic and reaches the required depth
- chapter starts and section breaks are correct
- body font, size, spacing, and first-line indent are consistent
- abstract and keywords match the template rules
- figure/table numbering and captions are correct
- table style meets the template, including three-line tables when required
- references are consistent
- in-text citations are correctly formatted
- no citation markers remain inside headings
- no blank pages, blank paragraphs, stray borders, or obvious layout defects remain
- unresolved content-judgment issues are explicitly listed for the user
